Stacked · Centered 로고를 베이스로, 배경 컬러만 변경한 비교 시안입니다. 로고 스트로크는 페이퍼(#F5F3EC)로 반전, 옐로우 사선(#F0B90B)은 브랜드 가이드에 따라 유지했습니다.

01 / Green

A · Emerald
Brand Emerald Green

인상 — 차분하고 신뢰감이 가장 강한 안인 동시에 브랜드 컬러 팔레트 안에서 일관성이 유지됩니다. 명리·자본 시장 분야에 어울리는 '안정·통찰'의 톤. 페이퍼 화이트(#F5F3EC)와 옐로우(#F0B90B)의 콘트라스트가 충분해 어두운 매장 조명에서도 가독성이 좋습니다. 가장 추천하는 안.
02 / Red

B · Deep Wine

C · Classic Red
Classic Signage Red

인상 비교
  • 딥 와인(#9B2C2C) — 와인바·서재·고급 한정식 같은 무게감. 옐로우 사선이 살짝 묻히지만 우아한 톤. 빨강을 쓰되 점잖게 가고 싶을 때 적합.
  • 클래식 레드(#B83227) — 발색이 강해 길거리에서 바로 인지되는 톤. 옐로우와의 콘트라스트가 활성화돼 활기차지만, 명리·금융 분야로는 다소 캐주얼·외향적인 인상.
⚠ 브랜드 가이드 충돌 가능성 — 브랜드 시트 06번 항목에 "1939년 MGM 영화의 시각 요소(루비 슬리퍼의 빨간색)는 사용 금지 — 워너브라더스 IP" 조항이 있습니다. 위 두 가지 빨강은 모두 의도적으로 채도·명도를 낮춰 루비 슬리퍼 톤을 피했으나, 빨강 자체가 「오즈의 마법사」 IP와 시각적으로 연상될 가능성을 완전히 배제하긴 어렵습니다. 간판으로 확정 시 상표 가이드 재검토 권장.

04 / Production Spec

제작 사양 (실내 간판 권장값)

권장 사이즈
가로 600 × 세로 450 mm (4:3) — 매장 정면 벽 또는 입구 측면 부착. 더 작게는 가로 400 mm까지 가독성 유지.
소재
1순위: 무광 아크릴 + UV 인쇄 (5T~8T). 2순위: 무광 코팅 메탈 + 실크 인쇄. 광택 소재는 옐로우 사선의 채도가 죽으니 지양.
색상 (CMYK / Pantone 근사값)
Emerald #00735E → Pantone 3298 C / C100 M0 Y65 K35
Wine #9B2C2C → Pantone 7621 C / C25 M90 Y85 K25
Classic Red #B83227 → Pantone 1797 C / C15 M95 Y95 K5
Yellow #F0B90B → Pantone 7549 C / C0 M30 Y100 K0
Paper(스트로크/텍스트) #F5F3EC → C3 M3 Y10 K0
여백
로고 기준 'O' 반지름 이상의 안전 여백 유지. 위·아래 11% / 좌·우 11% 패딩 권장(아트워크에 반영됨).
조명
매립등 또는 위에서 내려오는 워시 라이트 권장. 백라이트(LED 채널) 적용 시 옐로우 사선만 별도 발광 채널로 가공하면 야간 식별력↑.
마운트
네 모서리 스페이서 마운트(스테인리스 13 mm) — 벽면에서 살짝 띄워 그림자가 지도록. 직접 부착보다 고급 인상.
